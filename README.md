\# KB Bookshelf



KB Bookshelf is a full-stack app to manage your books, built with React + Vite on the frontend and Express + Google API on the backend. 

Search and add books to personalized bookshelves in a clean, responsive UI.



---



\## Features



\- Search books using the Google Books API

\- Add and delete books from bookshelves 

\- Full-stack separation: frontend (React + Vite) and backend (Express + Node.js)



---



\## Tech Stack



\- Frontend: React, Vite CSS written inline in components  

\- Backend: Node.js, Express, Google Books API  

\- Package Management: npm  



---



\## Installation



Follow these commands in your terminal to set up and run the KB Bookshelf app:



```bash

\# 1. Clone the repository

git clone https://github.com/katiebethbrady/kbBookshelf.git

cd kbBookshelf



\# 2. Install backend dependencies

cd server

npm install



\# 3. Install frontend dependencies

cd ../client

npm install



\# 4. Start the backend server

cd ../server

node index.js



\# 5. Start the frontend development server

cd ../client

npm run dev
